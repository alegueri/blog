## Why start a blog?

I've been building things for a while now — games, tools, apps — and I've never really written about any of it. That changes now.

This is going to be a place for me to:

- Document things I'm learning
- Share projects I'm working on
- Think out loud about design and code

## What's this built with?

- **Next.js 15** with the App Router
- **MDX** for writing posts (Markdown + React components)
- **Supabase** for comments and likes
- **Tailwind CSS** for styling
- Deployed on **Vercel**

Writing posts is as simple as dropping a `.mdx` file into `src/content/posts/` with some frontmatter at the top. That's it.

## What's next?

More posts, more projects. Stay tuned.
